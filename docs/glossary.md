# Краткий словарь

- **Токены:** кусочки текста, которыми оперирует модель.
- **Контекст:** вся информация, переданная модели в запросе.
- **Окно контекста:** максимальный объем текста, который модель учитывает.
- **Кэш токенов:** повторно используемый контекст, который стоит дешевле.
- **Патч/дифф:** изменения в виде списка добавлений и удалений.
- **Агент:** режим, где модель выполняет задачу автономно.
