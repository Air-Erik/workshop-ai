# Чек-лист экономии токенов

Перед запуском агента:
- Новый чат или summarize?
- Контекст минимально достаточный?
- Указаны только нужные файлы?
- Есть четкая цель и критерии?
- Ограничен размер задачи?

Во время работы:
- Делить на короткие итерации.
- Удалять лишние файлы из контекста.
- Сохранять успешные промпты для переиспользования.
