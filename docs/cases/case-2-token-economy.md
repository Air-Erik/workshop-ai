# Кейс 2: Экономия токенов в длинном диалоге

## Контекст задачи
- Команда ведет длинные диалоги с моделью.
- Лимиты выгорают за 1–2 недели.

## Плохой промпт
- Запрос с полной историей и лишними файлами.

## Исправленный промпт
- Новый чат + summarize + минимальный контекст.

## Результат
- Цена запроса снизилась, скорость выросла.

## Выводы
- Короткие итерации экономят токены и время.
