# Антипример и исправленный промпт (FastAPI + Kubernetes)

## Плохой промпт (антипример)
Сделай сервис на Python для FastAPI, который управляет k8s. Нужно создавать и
удалять деплойменты. Сделай быстро, как надо.

---

## Исправленный промпт (хороший пример)
Цель: добавить в FastAPI приложение сервис управления деплойментами Kubernetes.

Контекст:
- Используем Python 3.11 и FastAPI.
- Доступ к кластеру через k8s api.
- Нужны два эндпоинта: создать и удалить деплоймент.

Ограничения:
- Не менять существующую структуру проекта.
- Сервис и эндпоинты должны быть в отдельных файлах.
- Название namespace и имя деплоймента передаются в запросе.

Критерии результата:
- POST `/deployments` создает деплоймент.
- DELETE `/deployments/{name}` удаляет деплоймент.
- Возвращаются понятные ответы и обработка ошибок.

Формат ответа:
- Дай план изменений.
- Затем дай патч с кодом.
